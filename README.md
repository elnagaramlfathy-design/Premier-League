# Premier-League

📊 Premier League Data Analysis

📌 Project Overview

This project focuses on analyzing Premier League data using Microsoft Excel.
The goal is to clean, transform, and extract insights from raw football data to support better understanding of team and player performance.

⸻

🎯 Objectives
	•	Clean and preprocess raw data
	•	Analyze team performance
	•	Identify top players and statistics
	•	Create clear and interactive insights

⸻

🛠️ Tools Used
	•	Microsoft Excel
	•	Power Query (for data cleaning & transformation)
	•	Pivot Tables
	•	Charts & Visualization

⸻

🔄 Data Cleaning Process
	•	Removed duplicates
	•	Handled missing values
	•	Standardized data format
	•	Transformed data using Power Query

⸻

📈 Key Insights
	•	Top performing teams based on points
	•	Highest goal scorers
	•	Comparison between teams
	•	Trends in wins, losses, and draws

⸻

📊 Dashboard / Visualization
	•	Interactive charts
	•	Performance comparison
	•	Summary KPIs

⸻

🚀 Skills Demonstrated
	•	Data Cleaning
	•	Data Analysis
	•	Data Visualization
	•	Problem Solving
	•	Attention to Detail

⸻

📁 Files Included
	•	premier league.xlsx → Raw & cleaned dataset

⸻

💡 Conclusion
This project helped in improving data analysis skills and understanding how to turn raw data into meaningful insights.
